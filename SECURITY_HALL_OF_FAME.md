# Security Hall of Fame

> We credit security researchers who report vulnerabilities responsibly. With their consent, we list them here.

There are no reports yet — be the first.

## How to get listed

Report a vulnerability via [SECURITY.md](./SECURITY.md). If your report leads to a fix, you can choose to be credited in the fix commit and (with your consent) in this file.

We follow the [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) flow. Hall of Fame entries are added at the maintainer's discretion, with reporter consent.

## Recognition levels

| Level | Trigger |
|---|---|
| 🥉 Bronze | Valid report, no fix required (e.g. hardening suggestion already in place) |
| 🥈 Silver | Valid report leading to a fix shipped in a patch release |
| 🥇 Gold | Critical-severity report leading to a same-day fix and CVE |

We do not currently run a paid bug bounty program. This is a hobby project.
